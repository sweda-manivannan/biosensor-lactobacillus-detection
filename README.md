Biosensor for Probiotic Bacterial Detection

## Project Type
Academic Research Project | Biosensor Development | Product Concept Design

## Overview
Designed and developed a dual-mode biosensing platform for detecting *Lactobacillus agilis*, integrating a colorimetric detection system with CRISPR-Cas12a-based molecular confirmation.

## Objective
To develop a rapid, low-cost, and user-friendly diagnostic tool for probiotic detection in food and industrial applications.  

**Impact:** Enables faster, accessible quality control in probiotic manufacturing and food safety monitoring.

## Key Innovation
- Dual detection system:
  - Colorimetric biosensor using pH-responsive natural dyes
  - CRISPR-Cas12a-based molecular confirmation
- Combines visual detection with molecular specificity
- Designed for real-world usability and scalability

##  Methodology

### 1. Bacterial Standardization
- Developed OD vs CFU calibration curve
- Established reproducible bacterial concentration for biosensor testing

### 2. Colorimetric Biosensor Design
- Developed bioink using:
  - Sodium alginate
  - Cellulose nanocrystals
  - Anthocyanin (natural pH indicator)
- Observed visible color changes due to bacterial metabolic activity

### 3. Molecular Detection
- Designed PCR primers targeting the 16S–23S ISR region
- Validated amplification using gel electrophoresis
- Performed in silico validation using BLAST

### 4. CRISPR Integration (Concept Development)
- Designed workflow for Cas12a-based detection system
- Proposed dual-layer detection approach for enhanced specificity

## Key Results
- Established linear OD–CFU relationship for accurate bacterial quantification
- Demonstrated clear pH-dependent color change in biosensor system
- Successfully amplified target DNA region (~137 bp)
- Developed a scalable and reproducible detection workflow
